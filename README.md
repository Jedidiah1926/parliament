# DATANET: PARLIAMENT SIMULATION

> 레트로 터미널 스타일의 의회 의석 시각화 및 표결 시뮬레이터

---

## 개요

DATANET은 브라우저에서 바로 실행되는 의회 시뮬레이션 도구입니다. 양원제·단원제 전환, 정당 및 연정 관리, 법안 작성, 실시간 표결 집계를 지원하며, 냉전 시대 데이터 터미널에서 영감을 받은 CRT 모니터 감성의 UI로 구성되어 있습니다.

한국어 버전과 영어 버전이 제공되며, 두 버전은 동일한 저장 파일 형식(v2)을 사용합니다.

| 파일 | 언어 |
|------|------|
| `index.html` | 영어 |
| `tno.html` | 영어 |
| `tno_kor.html` | 한국어 |
| `beta_tno_kor.html` | 한국어 |

## 기능

### ⚙ 의회 설정

**이념 탭**
- 이념을 추가·삭제·순서 변경 가능 (좌우 스펙트럼)
- 정당은 기본적으로 이념 순서에 따라 자동 정렬됨

**하원 / 상원 탭**
- 탭 상단에서 의원실 이름과 총 의석 수 설정
- 정당별 이름, 색상, 의석 수, 이념 설정
- ▲ ▼ 버튼으로 정당 순서 수동 변경 가능
- 수동 정렬 중에는 상태 표시, `↺ 자동정렬` 버튼으로 이념 순 복귀
- 하원·상원 연동(LINK) 체크박스로 각 의원실 참여 여부 독립 설정
- 무소속 의원 슬롯 별도 추가 가능

**연정 탭**
- 여러 정당을 묶어 연정 구성, 연정 이름·색상 설정
- 연정 또는 단독 정당을 집권 세력으로 지정
- 집권 세력 의석은 캔버스에서 금색으로 강조 표시

---

### ⚖ 입법 절차

**법안 제출 탭**
- 법안 제목과 내용 작성 후 등록
- 대기 중인 법안만 카드 형태로 표시
- 표결이 완료된 법안은 자동으로 이 탭에서 사라지고 기록 탭으로 이동

**표결 탭**
- 드롭다운에서 심의할 법안 선택
- 투표 입력 모드 선택: 찬성(초록) / 반대(빨강) / 기권(회색) / 초기화
- 캔버스의 의원 원을 클릭해 개별 투표 입력
- 같은 모드로 다시 클릭하면 해당 의원 투표 초기화
- 정당 일괄 투표: 특정 정당 전체에 한 번에 투표 적용, 하원·상원·양원 선택 가능
- 하원 표결 먼저 확정해야 상원 표결 버튼 활성화
- 하원이 부결하면 상원은 자동으로 미상정 처리

**기록 탭**
- 가결·부결 완료된 법안 아카이브 (최신순)
- 가결 법안은 초록 테두리, 부결 법안은 빨간 테두리
- 각 의원실별 득표 수(찬성/반대/기권) 기록 표시

---

### 🖥 캔버스 시각화

- 반원형 의석 배치도, 정당 색상으로 구분된 점(dot)
- 점 테두리 = 당적 색상 / 점 내부 = 투표 상태 (투표 시)
- 집권 세력 강조 활성 시 해당 의석에 금색 글로우 효과
- 마우스 호버 시 툴팁: 정당명 · 이념 · 투표 상태 표시
- 창 크기 변경 시 자동으로 다시 그림

---

### 양원제 표결 흐름

```
법안 제출
    │
    ▼
하원 표결 ──── 부결 ──▶ 상원 미상정 → 최종 부결
    │
   가결
    │
    ▼
상원 표결 ──── 부결 ──▶ 최종 부결
    │
   가결
    │
    ▼
  최종 가결
```

## 저장 / 불러오기

**SAVE** 버튼을 누르면 현재 상태 전체를 `.json` 파일로 저장합니다.

저장 항목:
- 양원제·단원제 설정
- 각 의원실 이름 및 총 의석 수
- 이념·정당·연정 전체
- 정당 정렬 모드 (자동/수동)
- 법안 전체 및 표결 기록
- 현재 투표 입력 상태
- 열려 있던 탭 위치

**LOAD** 버튼으로 저장 파일을 불러옵니다. 한국어 버전과 영어 버전은 저장 파일이 완전히 호환됩니다.

> 구버전(v1) 저장 파일도 하위 호환으로 불러올 수 있습니다.

## 실행 방법

별도 설치나 빌드 과정 없이 HTML 파일을 브라우저에서 바로 열면 됩니다.

```
parliament_sim.html       ← 한국어 버전
parliament_sim_en.html    ← 영어 버전
```

Chrome, Firefox, Edge 최신 버전에서 동작합니다. 최초 실행 시 폰트를 외부 CDN에서 불러오므로 인터넷 연결이 필요합니다.

---

## 조작 방법

| 동작 | 설명 |
|------|------|
| 캔버스 클릭 | 현재 모드로 해당 의원 투표 입력 |
| 같은 모드로 다시 클릭 | 해당 의원 투표 초기화 |
| 정당 일괄 투표 버튼 | 해당 정당 전체 의석에 일괄 적용 |
| ↺ 자동정렬 | 이념 순으로 정당 순서 재정렬 |
| PROTOCOL EXECUTE | 캔버스 재렌더링 및 통계 갱신 |

## 디자인
